HOW TO: Install New Leadership Team

Last Updated: November 2025

This guide covers the complete process of installing a new leadership team at the start of a term using the Leadership Transition Portal.

Leadership Transition Portal

https://mail.sbnewcomers.org/admin/

This web-based tool automates leadership transitions by updating both Wild Apricot and email forwarding in a single workflow. Use this tool instead of manual updates.

Repository: sbnctech/sbnc-leadership-transition

Timing: This process should be completed on or immediately after the first day of the new term (Feb 1 or Aug 1). Coordinate with outgoing leadership to ensure a smooth handoff.

1. What the Portal Does

The Leadership Transition Portal automates the tedious parts of installing new leadership:

Portal Feature What It Does
Email Forwarding Management View, add, and remove email forwarders without cPanel access
Leadership Transitions Update multiple positions at once with validation
Wild Apricot Integration Automatically updates WA Admin contacts when changing leadership
Transition Spreadsheet Generates a spreadsheet of all positions for record-keeping
Google Drive Integration Archives transition documents
Why use the portal? Manual updates require logging into three different systems (Wild Apricot, cPanel, Website). The portal coordinates WA and email changes together, reducing errors and ensuring consistency.

3. Using the Leadership Transition Portal

1. Go to the Leadership Transition Portal
https://mail.sbnewcomers.org/admin/

2. Log in with your admin credentials
(See CREDENTIALS.md for login information)

3. Navigate to Leadership Transitions

4. For each position being changed:

  • Select the position from the list
  • Enter the new officer's name
  • Enter the new officer's personal email address
  • Click Update

5. Review the changes in the confirmation screen

6. Click Apply All Changes to execute the transition

Tip: The portal shows a preview of changes before applying them. Review carefully to ensure names and emails are correct.

4. What Happens Behind the Scenes

When you submit changes through the portal, it automatically:

Wild Apricot Updates:
  1. Finds the Admin contact for the position (e.g., "President Event_Chair")
  2. Updates the "Board or Chair member names" custom field
  3. Updates the "Board or Chair member emails" custom field
Email Forwarding Updates:
  1. Removes the old email forwarder (e.g., president → old_email@gmail.com)
  2. Creates a new forwarder (e.g., president → new_email@gmail.com)
  3. Updates take effect immediately
Logging:
  1. Records who made the change and when
  2. Archives the transition in Google Drive
  3. Generates a transition spreadsheet for records

5. Testing the Installation

After using the portal, verify everything worked correctly:

Test 1: Email Forwarding

1. Send a test email to each role address you updated
(e.g., president@sbnewcomers.org, treasurer@sbnewcomers.org)

2. Have the NEW officer confirm they received the email

3. Have the OLD officer confirm they did NOT receive the email

Expected Result: Each role email arrives ONLY at the new officer's personal email within 1-2 minutes.

Test 2: Wild Apricot Data Updated

1. Log into Wild Apricot Admin: sbnewcomers.wildapricot.org/admin

2. Go to Contacts > Filter by "Admins"

3. Open each position contact (e.g., "President Event_Chair")

4. Verify the custom fields show the new officer's name and email

Expected Result: The "Board or Chair member names" and "Board or Chair member emails" fields show the new officer's information.

Test 3: Chatbot Query (After Nightly Sync)

1. Wait for the nightly database sync (runs at 2 AM), OR ask Technology Chair to run a manual sync

2. Go to the CHAD chatbot: https://mail.sbnewcomers.org/chatbot/

3. Ask: "Who is the president?" (or other positions)

4. Verify the chatbot returns the NEW officer's name

Expected Result: The chatbot shows the new officer's name for each position.

Test 4: Portal Email Dashboard

1. In the Leadership Transition Portal, go to Email Management

2. Find the role email (e.g., "president")

3. Verify it shows forwarding to the correct personal email

6. Update the Website (Manual Step)

Note: The portal does NOT update the website. This step must be done manually in Wild Apricot.

1. In Wild Apricot Admin, go to Website > Site pages

2. Find and edit the Board/Leadership page

3. Update each officer's information:

  • Name
  • Photo (if used)
  • Keep the role email (e.g., president@sbnewcomers.org) - don't use personal emails

4. Save and Publish

5. Visit the public website and verify the page shows correct information

Certain positions require access to specific accounts. Handle these separately:

Position Accounts Needing Access

Treasurer Wild Apricot (finance), Bank accounts
Technology Chair Leadership Portal, Server access, all technical accounts
Activities VP Jotform (SBNCActivities)
Communications Newsletter system, Social media

  • Complete list of all accounts and current credentials
  • Password rotation procedures during transitions
  • How to add/remove Wild Apricot admin access

8. Final Verification Checklist

Portal Completed:
  • All board positions updated via Leadership Transition Portal
  • Portal shows correct email forwarding for all positions
  • Transition spreadsheet generated and archived
Testing Completed:
  • Test emails sent to all role addresses
  • New officers confirmed receipt of test emails
  • Old officers confirmed they did NOT receive test emails
  • Wild Apricot Admin contacts verified (names and emails correct)
  • Chatbot returns correct names (after sync)
Manual Steps Completed:
  • Website leadership page updated and published
  • Website displays role emails (not personal emails)
  • Wild Apricot admin access granted to new officers who need it
  • Wild Apricot admin access removed from outgoing officers
  • Passwords rotated for shared accounts
  • Credential handoff completed for each position
Troubleshooting: If something doesn't work as expected:
  • Check the portal's change log for errors
  • Verify the email address was typed correctly (no typos)
  • For chatbot issues, wait for nightly sync or request manual sync
  • Contact technology@sbnewcomers.org for help
