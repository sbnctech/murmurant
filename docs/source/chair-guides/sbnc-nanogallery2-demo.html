How Tags Work

Each photo has multiple tags in the data-ngtags attribute:

  • Event name: FallPicnic2024, HolidayParty2023, etc.
  • Date: Oct2024, Nov2024, Dec2023 (from EXIF)
  • Location: StearnWharf, ElksLodge, SandpiperGolf (from GPS)
  • Activity group: HappyHikers, GolfGroup, WineClub
  • People: JohnSmith, MaryJohnson, etc. (from face recognition)

AI Processing Pipeline (Proposed)

  1. Extract EXIF: Get date/time/GPS from each photo
  2. Match to Events: Use GPS + date to identify which SBNC event
  3. Face Detection: Find faces in each photo
  4. Face Recognition: Match against registered attendee photos
  5. Generate Tags: Create tag list for each photo
  6. Build Gallery JSON: Output data for nanogallery2
