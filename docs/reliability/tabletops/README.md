# Murmurant - Tabletop Scenarios (Pre-Deployment)

Status: Canonical Tabletop Set
Applies to: Pre-deployment readiness and incident rehearsal
Last updated: 2025-12-21

Tabletop scenarios are structured, discussion-based rehearsals used to validate:
- Guarantees
- Runbooks
- Human decision-making
- Escalation paths
- Stop / resume criteria

These scenarios are documentation-only.
They MUST NOT trigger real actions or automation.
